Учебный проект по анализу данных с тяжёлыми хвостами.
Состоит из:
1) Папки Latex с отчётом о проделанной работе и файлами необходимыми для компеляции отчёта LaTex.
2) Папки Graphics с сохранёнными графиками работы програмы.
3) Папки Materials + Data в которой хранятся предоставленные материалы и данные для обработки.
4) Файла Example_Mikhail.xlsx формата Excel с выборкой данных 7 столбца для обработки.
5) Файла Markovich_Data_Analysis.ipnb с программой.
