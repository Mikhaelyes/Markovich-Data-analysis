Учебный проект по анализу данных с тяжёлыми хвостами.
Состоит из:
1) Файла Отчёт.pdf с отчётом о проделанной работе.
2) Файла Markovich_Data_Analysis.ipnb с программой.
3) Папки Latex с отчётом о проделанной работе и файлами необходимыми для компеляции отчёта LaTex.
4) Папки Graphics с сохранёнными графиками работы програмы.
5) Папки Materials + Data в которой хранятся предоставленные материалы и данные для обработки.
6) Файла Example_Mikhail.xlsx формата Excel с выборкой данных 7 столбца для обработки.
